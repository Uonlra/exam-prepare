# NOTE

## C++ 模板

1.  `#include <bits/stdc++.h>`：包含几乎所有常用头文件（OJ 上常用写法）。
2. `using namespace std;`：使用标准命名空间，避免每次写 `std::`。
3. `ios::sync_with_stdio(false); cin.tie(nullptr);`：IO 加速。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    cout << "n = " << n << "\n";

    return 0;
}
```

## 一维数组

```cpp
//用简单数字来存一串数并遍历
int a[1000];
int n;
cin >> n;
for (int i = 0; i < n; i++) cin >> a[i];

int sum = 0;
for (int i = 0; i < n; i++) sum += a[i];
cout << sum << "\n";
```
