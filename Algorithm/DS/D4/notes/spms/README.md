# 📚 学生成绩管理系统 (SPMS)

## 🚀 启动方式

项目中提供了三种启动脚本，任选其一：

### 1️⃣ **启动SPMS.bat** (推荐新手使用)
- **文件**: `启动SPMS.bat`
- **使用方式**: 直接双击运行
- **特点**: 
  - ✅ 显示启动日志
  - ✅ 简单易懂
  - ✅ 适合初学者

### 2️⃣ **启动SPMS.vbs** (推荐日常使用)
- **文件**: `启动SPMS.vbs`
- **使用方式**: 直接双击运行
- **特点**:
  - ✅ 隐藏命令窗口
  - ✅ 界面更整洁
  - ✅ 推荐日常使用

### 3️⃣ **启动SPMS.ps1** (推荐高级用户)
- **文件**: `启动SPMS.ps1`
- **使用方式**: 
  ```powershell
  .\启动SPMS.ps1
  ```
  或右键选择"使用 PowerShell 运行"
- **特点**:
  - ✅ 现代化脚本
  - ✅ 彩色输出
  - ✅ 适合开发者

## 📖 快速开始

### 方式一：简单启动（推荐）
1. 双击 `启动SPMS.bat` 或 `启动SPMS.vbs`
2. 系统会自动打开浏览器并加载UI
3. 开始使用系统

### 方式二：手动启动
1. 用浏览器打开 `ui.html` 文件
2. 即可使用完整的学生管理系统

### 方式三：命令行启动
```bash
# CMD 命令行
start ui.html

# PowerShell
PS> Start-Process ui.html
```

## 💡 使用技巧

### 1. 添加学生
- 学号：10位数字（如：2020010001）
- 班级：1-20 的整数
- 姓名：大写字母开头，不含空白字符
- 成绩：0-100 的整数

### 2. 删除学生
- 可以输入学号或名字的任意一个
- 如果多个学生匹配，会全部删除

### 3. 查询学生
- 显示匹配学生的完整信息
- 包括排名、成绩、总分和平均分

### 4. 统计分析
- 显示各科平均分
- 显示及格/不及格人数
- 显示各种组合的及格人数

### 5. 学生排名
- 自动按总分从高到低排序
- 相同分数并列排名
- 显示排名和平均分

## 🐛 故障排查

### 问题1：启动脚本无反应
**解决方案**:
- 确保 `ui.html` 在同一目录下
- 尝试直接用浏览器打开 `ui.html`

### 问题2：浏览器没有打开
**解决方案**:
- 手动用浏览器打开 `ui.html` 文件
- 确保已安装浏览器（Internet Explorer 除外）

### 问题3：PowerShell脚本拒绝执行
**解决方案**:
```powershell
# 以管理员身份运行 PowerShell，执行：
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## 📝 数据持久化说明

⚠️ **重要**: 当前版本的UI使用浏览器内存存储数据，刷新页面后数据会丢失。

如果需要持久化存储，可以：
1. 使用后端服务器搭配数据库
2. 使用浏览器的 LocalStorage API
3. 使用原始的 C++ 命令行程序时数据持久化
