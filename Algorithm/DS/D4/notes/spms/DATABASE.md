# 📦 SPMS 数据持久化方案

## 概述

SPMS 学生成绩管理系统现已支持本地数据持久化，采用 **浏览器 LocalStorage** 方案，确保数据在浏览器中持久保存。

---

## 🗄️ 当前数据持久化方案

### 方案一：浏览器 LocalStorage（默认方案）✅

**特点：**
- ✅ 自动保存，无需手动操作
- ✅ 数据存储在浏览器本地
- ✅ 即使关闭浏览器，数据也会保留
- ✅ 容量约 5-10MB（足够存储数千个学生）
- ✅ 跨页面同步（同一浏览器的不同标签页）

**工作原理：**
1. 页面加载时自动从 LocalStorage 读取数据
2. 每次操作（添加、删除、修改）后自动保存
3. 页面卸载前自动保存数据
4. 数据采用 JSON 格式存储

**存储位置：**
- Chrome: `%AppData%\Local\Google\Chrome\User Data\Default\Local Storage`
- Firefox: `~/.mozilla/firefox/*/storage/default`
- Edge: `%AppData%\Local\Microsoft\Edge\User Data\Default\Local Storage`

## 💾 数据导入导出

### 导出数据为 JSON 文件

1. 在 UI 界面点击：**💾 导出数据**
2. 浏览器会下载一个 `spms_data_xxx.json` 文件
3. 该文件包含所有学生的完整信息

**导出文件格式：**
```json
[
  {
    "sid": "2020010001",
    "cid": 1,
    "name": "Alice",
    "scores": [85, 90, 88, 92]
  },
  {
    "sid": "2020010002",
    "cid": 1,
    "name": "Bob",
    "scores": [75, 80, 78, 85]
  }
]
```

### 导入数据从 JSON 文件

1. 在 UI 界面点击：**📂 导入数据**
2. 选择之前导出的 `.json` 文件
3. 系统会加载并保存这些数据

## 🔧 数据管理功能

| 功能 | 按钮 | 说明 |
|-----|------|------|
| 导出数据 | 💾 导出数据 | 将所有学生数据导出为 JSON 文件 |
| 导入数据 | 📂 导入数据 | 从 JSON 文件导入数据 |
| 清空数据 | 🗑️ 清空数据 | 永久删除所有数据（可恢复） |
| 手动保存 | 💾 保存 | 手动触发数据保存 |

## 🛡️ 数据备份建议

### 定期备份
1. 每周至少导出一次数据
2. 保存多个备份副本
3. 将备份存储在安全位置（云盘、外部硬盘等）

### 备份文件命名
```
spms_data_2024-03-23.json    # 按日期命名
spms_data_backup_v2.json     # 按版本命名
spms_data_final.json         # 最终版本
```

## 🔄 数据恢复流程

### 如果数据丢失

1. **清空浏览器数据后恢复：**
   - 打开 UI 界面
   - 点击 **📂 导入数据**
   - 选择之前导出的 json 文件
   - 数据会被立即恢复

2. **切换浏览器/电脑：**
   - 将导出的 json 文件复制到新环境
   - 打开 UI 界面
   - 点击 **📂 导入数据** 导入文件

## ⚠️ 注意事项

### 存储限制
- **浏览器缓存清空**：如果清空浏览器缓存或 Cookie，LocalStorage 可能丢失
  - 定期导出是最好的防护措施

### 隐私浏览模式
- **无痕浏览**：在无痕模式下数据不会持久化
  - 关闭浏览器后数据会丢失

### 跨浏览器
- 不同浏览器之间数据不同步
- 使用导入导出功能在浏览器间转移数据

### 浏览器更新
- 浏览器更新通常不会丢失 LocalStorage 数据
- 但建议提前导出备份

## 📝 常见问题 (FAQ)

### Q: 数据存储在哪里？
**A:** 数据存储在浏览器的 LocalStorage 中，位置随浏览器不同而不同。

### Q: 如果我换电脑了怎么办？
**A:** 在原电脑上导出 json 文件，然后在新电脑的浏览器中导入该文件。

### Q: 数据会自动保存吗？
**A:** 是的，每次操作后都会自动保存到 LocalStorage。

### Q: 可以在多个浏览器中使用吗？
**A:** 可以，但每个浏览器的数据是独立的。可以通过导入导出功能同步数据。

### Q: 如何确保数据安全？
**A:** 定期导出 json 文件进行备份，保存在安全的位置。

### Q: LocalStorage 有容量限制吗？
**A:** 通常 5-10MB，足以容纳数千个学生记录。

### Q: 如何清空所有数据？
**A:** 点击 **🗑️ 清空数据** 按钮，确认清空即可。
